What is CSS?

  • CSS stands for Cascading Style Sheets.
  • It is used to style and design web pages created with HTML.
  • Controls colors, fonts, spacing, layouts, responsiveness,
    animations, and much more.

Types Of CSS

Inline<style as an attribute>

Internal<style as an tag>

External<links file to the HTML Document>

CSS Selectors

  • Element Selector ‚Üí p { color: green; }
  • Class Selector ‚Üí .box { background: yellow; }
  • ID Selector ‚Üí #title { font-size: 24px; }
  • Universal Selector ‚Üí * { margin: 0; padding: 0; }
  • Group Selector ‚Üí h1, h2, h3 { color: blue; }
  • Descendant Selector ‚Üí div p { color: gray; }

CSS Properties

Color & Background

  • color ‚Üí Sets text color.
  • background-color ‚Üí Sets background color.
  • background-image ‚Üí Adds an image as background.

body {
background-color: lightgray; /* page background */
}
h1 {
color: navy; /* text color */
}
div {
background-image: url("bg.jpg"); /* background image */
background-size: cover; /* cover full area */
}

Text Properties

  • font-size ‚Üí Controls text size.
  • font-family ‚Üí Changes font style.
  • font-weight ‚Üí Boldness (normal, bold, 100‚Äì900).
  • text-align ‚Üí Aligns text (left, center, right, justify).
  • text-decoration ‚Üí Underline, line-through, none.

p {
font-size: 18px;
font-family: Arial, sans-serif;
font-weight: bold;
text-align: center;
text-decoration: underline;
}

Box Model

Container

  • width / height ‚Üí Size of the box.
  • padding ‚Üí Space inside border.
  • border ‚Üí Line around the box.
  • margin ‚Üí Space outside border.

div {
width: 200px;
height: 100px;
padding: 10px; /* inner space */
border: 2px solid black;/* border line */
margin: 20px; /* outer gap */
}

Positions

  • position ‚Üí Defines element‚Äôs position.
  • static (default)
  • relative (move based on itself)
  • absolute (relative to parent)
  • fixed (always in viewport)
  • sticky (sticks when scrolling)

.box {
position: absolute;
top: 50px;
left: 100px;
}

Layouts

Flex Box-2D

  • display: flex; ‚Üí activates flexbox.
  • justify-content ‚Üí horizontal alignment.
  • align-items ‚Üí vertical alignment.
  • flex-direction ‚Üí row or column.

.container {
display: flex;
justify-content: center; /* center horizontally */
align-items: center; /* center vertically */
}

Grid Box-3D

  • display: grid; ‚Üí activates grid.
  • grid-template-columns ‚Üí defines columns.
  • grid-template-Rows ‚Üí defines rows.
  • gap ‚Üí space between items.

.grid {
display: grid;
grid-template-columns: 1fr 1fr 1fr;
gap: 10px;
}
