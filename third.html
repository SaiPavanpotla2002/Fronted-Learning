About Java-Script

What is JavaScript?

  • JavaScript (JS) is a programming language mainly used to make websites dynamic and interactive.
  • It runs directly in the web browser (Chrome, Edge, Firefox, Safari) or on servers using Node.js.
  • Together with HTML (structure) and CSS (style), JavaScript adds the behavior.

Why is JavaScript important?

  • It is the most widely used language for web development.
  • Can be used for frontend (browser apps) and backend (Node.js servers).
  • Powers websites, mobile apps, games, desktop apps, and APIs.
  • it makes web pages interactive and dynamic.

Features of JavaScript

  • Interpreted (runs directly in browser, no compilation needed).
  • Cross-platform (works on Windows, Mac, Linux, Mobile).
  • Event-driven (responds to user clicks, input, etc.).
  • Cross-platform (works on Windows, Mac, Linux, Mobile).

Uses of JavaScript

  • Frontend: animations, forms, dynamic UI
  • Backend: servers with Node.js
  • Mobile: React Native, Ionic
  • Desktop: Electron apps (like VS Code)
  • Data visualization: D3.js, Chart.js

What is a Variable

A variable is a container used to store data values (like numbers, text, objects, etc.) so you can use and update them in your program.

Functions in Java-Script

A function is a block of code that performs a specific task.
Functions help you reuse code, make programs modular, and keep code organized.

Syntax

function functionName(parameters) {
// code to be executed br return value; // optional
}

Example

function greet(name) {
return "Hello, " + name + "!";
}

console.log(greet("Sai")); // Output: Hello, Sai!
console.log(greet("Pavan")); // Output: Hello, Pavan!
